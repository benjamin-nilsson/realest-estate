# Realest Estates
A real-estate app.

### To run:
`npm start`

### To clone down and use:
1. Clone
2. Delete package-lock.json
3. Run on terminal: “npm install”
4. Run on terminal: “npm start”
5. Run “npm run server” on a separate terminal
